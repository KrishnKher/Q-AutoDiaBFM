# Q-Diff
Automatic Diagnosis of Quantum Software Bug Fix Motifs

Instructions to run the source code:

- git clone https://github.com/KrishnKher/Q-AutoDiaBFM
- Pass in two strings of code, one being the buggy and the other, the patched one to the bugInvestigator.classifyBugs method being called in [main.py](https://github.com/KrishnKher/Q-AutoDiaBFM/blob/main/src/main.py)
